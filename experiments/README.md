# Experiments

One folder per experiment: `NNNN-short-name/`. Copy `_TEMPLATE/` to start.

An experiment is not "run some code". It is: a hypothesis, a baseline, exactly
one changed variable, a pre-registered prediction, raw results, and an honest
conclusion. If a folder is missing any of those, it's not done.
